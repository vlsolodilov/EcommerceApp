# EcommerceApp

Приложение имитация интернет магазина. Использовано Api: [DummyJSON](https://dummyjson.com).
В приложении есть авторизация (логин: kminchelle, пароль: 0lelplR).

### Стек:
- Kotlin
- Single Activity
- XML
- Retrofit
- Gson
- Coroutines
- Navigation Architecture Component
- Viewpager 2
